# Taylor Johnson Professional Cleaning Services Website

## 🌐 Live Website

https://taylorsprocleaning.com

---

## 📌 Project Overview

Developed a responsive, production-ready business website for a local cleaning service to establish an online presence and streamline customer inquiries. The project focuses on clean UI design, performance, and integration of third-party services without a backend.

---

## 🚀 Key Features

* Responsive, mobile-first design for cross-device compatibility
* Custom domain integration with DNS configuration
* Contact form with serverless submission handling
* Optimized static asset delivery for fast load times
* HTTPS-enabled deployment for secure user interactions

---

## 🛠️ Technical Implementation

* Built using **HTML** and modern **CSS**
* Deployed via GitHub Pages with continuous deployment from the main branch
* Configured custom domain through Namecheap, including DNS records (A + CNAME)
* Integrated **Formspree** for backend-free form handling and email notifications
* Implemented SSL/TLS certificate provisioning for secure HTTPS access

---

## 🔧 Challenges & Solutions

* **Domain Configuration:** Resolved DNS propagation and GitHub Pages domain verification issues by correctly configuring A and CNAME records
* **Form Handling Without Backend:** Leveraged Formspree to enable reliable form submissions without building a custom server
* **Deployment Issues:** Diagnosed and resolved domain routing errors during initial setup

---

## 📈 Impact

* Delivered a professional web presence for a local business
* Improved accessibility for potential clients to learn about services and make contact
* Demonstrates ability to deploy and manage a full static site with custom domain configuration
